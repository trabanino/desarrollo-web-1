Esta es la manera menos recomendable de utilizar CSS dentro de un elemento HTML ya que cambiarlo resulta complicado. Lo recomendable es que en caso de ser necesario un estilo único en una página se utilice el style dentro del header como se hace en esta página para el background-color.
